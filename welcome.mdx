Lumi Mint Bot (Beta) is a desktop tool that helps you mint NFTs the moment a drop opens, across multiple wallets, without you having to sit refreshing a page and clicking "Mint" by hand.

It runs as a simple menu you navigate with your arrow keys and Enter, no coding required.

Built by [@cmonkey0650](https://x.com/cmonkey0650).

## How it works, in one paragraph

You tell the bot which wallets to use and which drop to target by editing one settings file. When you run the bot, you pick a platform, pick which wallets to mint with, pick the sale stage you're eligible for, and then choose a **mode**, for example "wait for the drop to open and mint automatically" or "just try to mint right now." The bot then handles the technical parts: checking eligibility, building the transaction, paying gas, and submitting it to the blockchain.

## Why a CLI, not a GUI or web app?

Lumi Mint Bot (Beta) is a command-line tool on purpose, not a limitation we plan to grow out of.

- **Faster to ship new modes.** Adding a new mint mode or platform to a menu is a small, low-risk change. Doing the same in a GUI means redesigning screens and layouts every time, which slows down how quickly new features can reach you, especially during the drops where timing matters most.
- **Safer than a web app.** Your private keys stay on your own machine and are never sent anywhere. A web app would need your private keys stored on a server to mint on your behalf, and that server becomes a single point of failure. If it's ever compromised, every user's keys are exposed at once.
