title: 'The Main Menu'
description: 'A tour of the top-level menu you see on launch'
---

Once your config is filled in and your license key is valid, launching the bot shows this menu:

```
Select mode:
  OpenSea
  Blever
  Transient
  Custom Contract
  Wallet Tools
  ↺  Reload Config
  Exit
```

<Frame caption="The main menu">
  <img src="/images/main-menu.png" />
</Frame>

Navigate with the **up/down arrow keys** and press **Enter** to select, in every menu throughout the bot.

| Option | What it does |
|---|---|
| **OpenSea** | Mint from an OpenSea SeaDrop collection. See [Minting Guide](/08-minting-guide). |
| **Blever** | Mint from a Blever.xyz drop. See [Minting Guide](/08-minting-guide). |
| **Transient** | Mint from a Transient.xyz drop. See [Minting Guide](/08-minting-guide). |
| **Custom Contract** | Mint from any other contract by building or discovering calldata. See [Custom Contract Mode](/10-custom-contract). |
| **Wallet Tools** | Allowlist checking, balance checking, and moving tokens/NFTs between your own wallets. See [Wallet Tools](/11-wallet-tools). |
| **↺ Reload Config** | Re-reads your settings file without restarting the bot. Use this after editing the config while the bot is open. |
| **Exit** | Quits the bot. |

<Tip>
  Press Ctrl+C at any point to back out to the previous menu, or to the main menu.
</Tip>
