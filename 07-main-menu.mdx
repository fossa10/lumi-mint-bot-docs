---

Once your config is filled in and your license key is valid, launching the bot shows this menu:

```
Select mode:
  OpenSea
  Blever
  Transient
  Custom Contract
  Wallet Tools
  ↺  Reload Config
  Exit
```

<Frame caption="Screenshot: the main menu">
  <img src="/images/placeholder-main-menu.png" />
</Frame>

Navigate with the **up/down arrow keys** and press **Enter** to select, in every menu throughout the bot.

| Option | What it does |
|---|---|
| **OpenSea** | Mint from an OpenSea SeaDrop collection. See [Minting Guide](/minting-guide). |
| **Blever** | Mint from a Blever.xyz drop. See [Minting Guide](/minting-guide). |
| **Transient** | Mint from a Transient.xyz drop. See [Minting Guide](/minting-guide). |
| **Custom Contract** | Mint from any other contract by building or discovering calldata. See [Custom Contract Mode](/custom-contract). |
| **Wallet Tools** | Allowlist checking, balance checking, and moving tokens/NFTs between your own wallets. See [Wallet Tools](/wallet-tools). |
| **↺ Reload Config** | Re-reads your settings file without restarting the bot — use this after editing the config while the bot is open. |
| **Exit** | Quits the bot. |

<Tip>
  Pressed Ctrl+C by accident? The bot catches it and brings you back to this main menu instead of crashing.
</Tip>
