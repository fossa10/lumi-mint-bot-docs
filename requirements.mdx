## A license key

You need an active license key before the bot will show its menu. See [Getting a License Key](/getting-license-key).

<Warning>
  A license key locks to the **first computer** it's used on. If you plan to move to a new PC, reinstall your OS, or use a virtual machine, read the machine-lock note on that page first.
</Warning>

## One or more funded wallets

You'll need at least one Ethereum-compatible wallet with:

- Enough of the chain's native token (ETH, etc.) to cover the mint price, if any
- Enough extra to cover **gas fees**, the network fee for submitting a transaction. Always keep more than you think you need; gas prices spike during popular drops.

See [Setting Up Your Wallets](/wallet-setup) for exactly how to give the bot access to your wallets.

## RPC endpoints

The bot talks to the blockchain through "RPC" servers. It ships with working public RPC URLs for Ethereum and Base already filled in, so most users don't need to do anything here.

If you want faster or more reliable RPCs, especially for competitive mints, see [RPC Setup](/rpc-setup) for how to get a paid RPC endpoint.

## Proxy (required for 3-5+ wallets)

A proxy can help avoid rate limits when checking eligibility across many wallets. See [Proxy Setup](/proxy-setup).

## VPS (optional, but highly recommended)

For lower latency and more reliable uptime than a home connection, some users run the bot on a VPS (virtual private server) instead of their own computer. It's also safer in one sense: a VPS is isolated from your personal and critical files and data, so it keeps bot separate from your day-to-day computer. See [VPS Setup](/vps-setup).

## Basic comfort editing a text file

You don't need to write any code, but you will need to open one settings file in a plain text editor and fill in some values. Every field is explained in [Configuring the Bot](/configuration).

<Tip>
  Basic editors like Notepad (Windows) or TextEdit (Mac) work, but [Notepad++](https://notepad-plus-plus.org/) (Windows) or [Sublime Text](https://www.sublimetext.com/) (Windows/Mac) are highly recommended. They highlight YAML syntax, which makes it much easier to spot a misplaced quote or indentation mistake in your settings file.
</Tip>
